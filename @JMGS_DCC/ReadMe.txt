// ドレスコード設定＆反映 MOD for MENTAIKO ARMA3 SERVER
//                                                           Create ITOM
//---------------------------------------------------------------------------
本スクリプトは ドレスコード（ユニフォーム、ベスト、ヘッドギア）の統一時に
設定値を自動的にセットするスクリプトMODです。

サーバーへ入った時のユニフォーム、ベスト、ヘッドギア、バックパックを指定でき、バックパックへアイテムを追加できます。


・「ドレスコード登録」でメニューを実行したユーザーのユニフォーム、ベスト、ヘッドギアを記憶

・「ドレスコード呼出」で記憶されたユニフォーム、ベスト、ヘッドギアを変更
   持っていたアイテムもそのまま（のはず）


導入方法
1. @TFAR_AutoSetting フォルダを Arma3のインストールフォルダへ入れる
2. @TFAR_AutoSetting\userconfig\TFAR_AutoSetting フォルダを Arma3のインストールフォルダの userconfig フォルダへ入れる

　以下のようになる
  - Arma 3
     - @DressCodeChanger
       - addons
          - DressCodeChanger.pbo
     - userconfig
       - DressCodeChanger
          - setting.hpp
